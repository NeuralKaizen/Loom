# decide

Record a technical **decision** with its context, the moment you make it — so future-you
can answer "why did we do X, and what did we rule out?" without archaeology.

Part of the [loom](../README.md) second-brain collection. Writes a lean ADR-style note
(decision · context · alternatives · tradeoff · consequences) to your vault's
`decisions/` folder, linked into the graph. Decisions form a chain — a new one can
supersede an older one.

## Triggers

- "vamos con X en vez de Y", "decidí usar X"

- "registrá / anotá esta decisión"

Not for trivial or easily-reversible choices — and if you're just *asking* which option
to pick, it answers first and offers to record it once you choose.

## Install

Ships with loom — see the [collection README](../README.md).
