# Topic color palette

Dark-mode-friendly accents (Catppuccin Mocha). thread assigns these to topics **in the
order listed**, skipping any color already taken in `~/.config/loom/colors.json`, and
never reshuffles one that's already assigned.

For `graph.json`, `color.rgb` is the hex as a **decimal integer** — compute it when
writing (e.g. `#89b4fa` → `9024762`).

1.  blue      `#89b4fa`
2.  green     `#a6e3a1`
3.  yellow    `#f9e2af`
4.  peach     `#fab387`
5.  mauve     `#cba6f7`
6.  red       `#f38ba8`
7.  teal      `#94e2d5`
8.  pink      `#f5c2e7`
9.  sky       `#89dceb`
10. lavender  `#b4befe`
11. sapphire  `#74c7ec`
12. flamingo  `#f2cdcd`

If the vault has more topics than colors, cycle back to the top — repeats are fine once
every color is in use.
